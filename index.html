Welcome to the Data Processing Project

\n

\n This project demonstrates a robust data processing pipeline for transforming raw data into structured insights.\n Using Python and Pandas, we process an Excel dataset, convert it to CSV, and then aggregate it into a JSON output.\n

\n

\n The entire process is automated via GitHub Actions, including linting with Ruff, execution of the data script,\n and deployment of the generated JSON output to GitHub Pages.\n

\n
\n

Key Files:

\n
    \n
  • execute.py: The Python script for data processing.
    • \n
  • data.xlsx: The initial raw data in Excel format.
    • \n
  • data.csv: The committed CSV version of the raw data.
    • \n
  • .github/workflows/ci.yml: GitHub Actions workflow for automation.
    • \n
  • result.json: The processed output (generated by CI, deployed to GitHub Pages).
    • \n

How it Works

\n

\n The core of this project lies in execute.py. This script reads the pre-converted data.csv,\n performs specific data aggregations (e.g., summing values by category), and then outputs the result\n into result.json.\n

\n

\n Our CI/CD pipeline ensures code quality and automated execution. On every push to the main branch,\n Ruff lints the Python code, execute.py runs to generate the latest result.json,\n and this JSON file is then published to GitHub Pages, making the processed data easily accessible.\n
